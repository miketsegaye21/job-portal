# Job Portal Application

A comprehensive full-stack job board platform designed to seamlessly connect employers with job seekers. 

## 🚀 Features

### For Job Seekers
- **Search & Filter:** Browse jobs by category, keywords, and location.
- **User Profiles:** Build and manage a professional seeker profile.
- **Applications:** Apply to listings easily and track application statuses in real-time.

### For Employers
- **Company Profiles:** Create and showcase a detailed company profile.
- **Job Management:** Post, edit, and delete job listings.
- **Applicant Tracking:** View incoming applications and manage candidates.

### Admin Features
- **User Management:** Oversee and manage both employer and seeker accounts.
- **Content Moderation:** Monitor and moderate job postings to ensure quality.

## 🛠️ Tech Stack
- **Frontend:** React (built with Vite)
- **Backend:** Node.js with Express.js
- **Database:** SQL (Custom schemas included for relational data management)

## 📦 Project Structure
- `/frontend` - Contains the React user interface and components.
- `/backend` - Contains the Node.js REST API, authentication logic, and database schemas.
- `/docs` - API documentation and ER diagrams.

## ⚙️ Local Setup

### Backend Setup
1. Navigate to the backend directory: `cd backend`
2. Install dependencies: `npm install`
3. Configure your environment variables by copying `.env.example` to `.env`.
4. Initialize the database using `src/database/schema.sql`.
5. Start the backend server.

### Frontend Setup
1. Navigate to the frontend directory: `cd frontend`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
